# DSTにおける銀河N体シミュレーション Rust実装のためのプログラマ指示書（DST独自部分のみ）

## 概要

本書は、双対時空理論（Double Spacetime Theory, DST）を用いた銀河シミュレーションを重力多体シミュレーションとしてRust言語で実現する際の、**DST特有の算出方法のみ**を記述したプログラマ向け指示書です。標準的なN体シミュレーションの実装方法（粒子データ構造、時間積分器、力の総和最適化など）には一切触れません。平坦回転曲線を暗黒物質なしで再現するためのDST独自の重力則と関連パラメータのみを対象とします。

## 1. S³上での座標・運動計算と三つ組Lnによる重力計算（DST独自モデル）

DSTモデルに基づき、以下のDST忠実アプローチを実装してください（平坦空間近似ではなく、より根本的なS³ + Lnモデル）：

- **物体の座標と移動**: S³上で計算。位置を単位四元数で表現。移動・回転は単位四元数の乗算により行う（S³上の測地線運動や回転を自然に表現）。四元数ライブラリを使用し、単位ノルムを維持。

- **重力計算**: 三つ組のLn（対数写像、3成分対応）で行う。
  1. 粒子iとjの相対位置をS³上の単位四元数 q_rel として取得。
  2. Ln変換（log map）により su(2) ≅ ℝ³ の3成分ベクトル v = Ln(q_rel) に射影（これにより大きさだけでなく方向も歪曲してくる）。
  3. 射影された3D空間で、固定R（観測データから決定した定数）と質量Mを用いた重力則を適用。方向はLn空間でのvの方向をそのまま使用（歪曲を妨げずに反映）。
  4. 得られた加速度を適切にS³に戻し（exp map）、四元数運動方程式に適用。

- **カイラリティーの乱れ**: 予想されるカイラリティーの乱れを妨げずに、そのまま算出してください。人為的なカイラリティー補正や向きの強制正規化を加えない。

- **銀河の渦の自然発生**: 方向歪曲＋大きさ歪曲の集団効果により、銀河の渦（スパイラル構造や渦状腕）が自然に発生すると想定。人工的な渦強制や特殊初期条件は不要。

- Rは観測データから決定した銀河全体の固定定数を使用（粒子ごと変更なし）。

- 全粒子からの寄与を総和し、バリオン質量分布のみから平坦回転曲線＋自然渦形成を再現。

- 単位四元数で表された位置が一回転を超えてもその角度を座標として維持してください。しかし、Lnについては計算（log map）がいきなり正反対になるように見えます。それで構いません。計算上の重力は、銀河半径の定数を超えた領域では斥力反転すると考えます。

これにより、DSTのS³投影図法の歪み（大きさ＋方向）を忠実に取り込み、カイラリティー乱れを許容した計算が実現します。

## 2. 銀河の半径 R の定義（定数として）

DSTの今回のシミュレーションでは、銀河の半径 R は**シミュレーション全体を通じて固定の定数**として定義します。R は5万光年で固定してください。

位置を定義する単位四元数が、180度回転する角度が R に対応します。1回転すれば、それは 2R の距離となります。

## 3. 検証ポイント

- 初期条件：バリオン質量分布のみ（暗黒物質ハローなし）。粒子の恣意的な回転運動もなし。
- 進化後、回転曲線 v(r) を測定。r ≈ 0.5R 〜 R の領域で平坦になることを確認（η(r)強化による）。
- 外縁で緩やかな上昇（r ≳ R）と、斥力反転を確認。
- これによりDST予測の「暗黒物質不要の平坦回転曲線」が再現されます。

以上がDST特有の計算方法です。この計算方法を適用することで、双対時空のS³周期性と投影図法の歪みが取り込まれたシミュレーションが実行可能になり、観測と一致する銀河シミュレーションが実現するかどうかの検証が実現します。
